# [VercelToChinese](https://github.com/GamblerIX/VercelToChinese)

## 简介

将 Vercel 的英文界面翻译为中文。词库与脚本逻辑分离维护，欢迎补充词库。

## 安装与启用

- 点击会触发脚本管理器安装：[VercelToChinese.user.js](https://raw.githubusercontent.com/GamblerIX/VercelToChinese/main/scripts/VercelToChinese.user.js)

## 开发

- 修改脚本：`scripts/VercelToChinese.user.js`
- 修改词库：`i18n/zh-cn.json`
