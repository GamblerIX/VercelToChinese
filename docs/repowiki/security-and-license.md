# 安全与许可证

## 安全注意事项

- 本项目脚本通过在浏览器端读取与替换 DOM 文本实现汉化，不需要也不应收集用户数据。
- 不要在词库或脚本中加入会输出敏感信息的日志，尤其是环境变量、令牌、请求头等。
- 若使用 `@require` 外部词库（例如 `old2.user.js`），需要意识到这引入了供应链依赖：
  - 外部脚本不可用会导致翻译失效
  - 外部脚本被篡改可能带来风险
  - 建议在发布说明中明确来源，并在故障排查中提供验证方式

## 许可证说明

- 仓库根目录的 LICENSE 为 GNU AGPLv3（Affero GPL v3）。
- 仓库内历史脚本头部存在不同的 `@license` 声明（例如 `GPL-3.0`、`MIT`），可能与仓库 LICENSE 不一致。

建议：

- 对外分发时尽量统一并明确“以仓库 LICENSE 为准”或“以分发脚本的 LICENSE 为准”，避免使用者误解。
- 若你计划合并/演进为单一脚本版本，建议同步整理脚本头部的 `@license` 字段，与仓库实际许可证保持一致。
